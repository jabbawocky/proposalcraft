ProposalCraft · Install

3 lines.
That's the install.

Open Claude Desktop config. Paste. Restart. ProposalCraft tools appear automatically.

~/Library/Application Support/Claude/claude_desktop_config.json
{
"mcpServers": {
"proposalcraft": {
"command": "npx",
"args": ["-y", "github:bradshawprojects/proposalcraft"]
}
}
}
✓
No API key. No account. No cloud. Your proposals stay on your machine. Works with Claude Desktop on macOS and Windows. Node.js is the only requirement — you already have it.
