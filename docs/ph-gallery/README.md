## Image 2 — animated HTML (no Mac recording needed)

`image2-core-loop.html` is a self-contained animation that runs entirely in the browser:

1. **Open** `image2-core-loop.html` in Chrome (or use the GitHub Pages URL once pushed)
2. **Start a screen recorder** — Mac's built-in Screenshot app (⌘+Shift+5), Loom, Kap, whatever you have
3. **Wait ~15 seconds** for one full loop to complete
4. **Stop recording** → save as GIF (drag MP4 to ezgif.com/video-to-gif if needed, optimize to <5MB)

Caption for PH: **"30 seconds. Ready to send."**

The animation loops automatically, so if the first take is off, just wait for the next loop.
