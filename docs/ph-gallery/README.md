## Image 2 — GIF (requires Mat)

**Record this with QuickTime on your Mac:**

1. Open Claude Desktop, paste the ProposalCraft JSON config if not already installed
2. Open QuickTime → File → New Screen Recording, select Claude Desktop window
3. Type: `draft a proposal for a Shopify redesign project, $15k budget, 3 months`
4. Let it run — show the tool call (`draft_proposal` activating), let the response render
5. Stop recording. Total runtime: 10–15 seconds
6. Convert to GIF: drag MP4 to ezgif.com/video-to-gif, optimize to under 5MB

Caption: **"30 seconds. Ready to send."**
